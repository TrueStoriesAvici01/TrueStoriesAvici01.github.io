Vim常用操作

Vim常用操作

[toc]

Vim中常用的命令

光标移动

移动方向 快捷键
j 向下移动一行
k 向上移动一行
h 向左移动一行
l 向右移动一行
0 移动到当前行首
$ 移动到当前行尾
]] 段首
[[ 段尾
G 最后一行
gg 第一行
nG 跳转到特定行

搜索

搜索方式 快捷键
/word 光标后搜索包含word的位置
?word 光标前搜索包含word的位置
n 搜索下一个结果
N 搜索前一个结果

复制、粘贴、删除

功能 快捷键
x,X 向后、向前删除一个字符
dd 删除光标所在行
ndd 从光标处向下删除n行
d$ 删除光标到所在行行尾
d0 删除光标到所在行行首
yy 复制所在行
nyy 向下复制n行
p,P 向后、向前粘贴
J 将所在行与下一行合并为一行
u 撤销前一个动作
ctrl + r 反撤销
. 重复上一个动作

保存

功能 快捷键
wq! 强制保存后退出
q! 强制退出不保存

可视化界面

多行缩进

  • 按键v进入可视化界面。
  • 通过j向下移动光标选中多行。
  • 按键<向前缩进，按键>向后缩进。

多行注释

  1. Ctrl + q进入viusal block模式。
  2. 通过j或k选中多行。
  3. 通过Ctrl + i进入insert模式。
  4. 输入输入的内容，通过Esc键退出编辑模式即可。
