基本数据类型

数据类型 解释
bool 布尔值
string 字符串
int 整数
int8 8位整数
int16 16位整数
int32 32位整数
int64 64为整数
uint 无符号整数
uint8 8位无符号整数
uint16 16位无符号整数
uint32 32位无符号整数
uint64 64位无符号整数
uintptr
byte 等价于uint8
rune 等价于int32
float32,float64 32、64位浮点数
complex32，complex64 32、64位复数

注意：int，uint和uintptr是根据平台决定采用的长度，32位操作系统为32位比特，64位操作系统为64比特。

零值

当一个变量定义时没有赋值，则会被分配“零值”。

  • 数值类型为0
  • 字符串为””
  • 布尔类型为false

类型转换

在go中，类型转换必须显式指定。通过T(v)可以将变量v转换为T类型。

类型推导

在定义变量时，若没有指定变量类型（通过:=或var a = )时，则变量的类型是由右值推导出来的。

  1. 当右值指定数据类型，则该变量的数据类型与右值保持一致。
  2. 若右值为非类型的数值常量，则根据常量的精度可能为int,float64或complex128。

常量

常量的声明类似于变量，使用关键字const。
常量可以为字符，字符串，布尔或数值。
**常量的声明不能使用:=**。

数值常量

一个没有指定类型的数值常量可以根据上下文决定对应的数据类型。
