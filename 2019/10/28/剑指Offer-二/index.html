  • 1.编程基础功(字符串处理)；2.并发控制；3.算法、复杂度；4.语言的基本概念。
  • 1.编程基础；2.计算机系统基础知识；3.算法及设计能力。
  • 1.对OS的理解：内存管理、文件操作、程序性能、多线程、程序安全；2.对于一门编程语言的掌握程度；3.算法和数据结构。

数据结构

  1. 数组和字符串用连续内存分别存储数字和字符。
  2. 链表和树需要大量的指针，需要留意鲁棒性。
  3. 栈与递归紧密相关。
  4. 队列也与广度优先遍历算法紧密相关。

数组

特点：
占据内存中的连续的一定空间并按照顺序存储数据；
创建数组需先指定数组的容量大小。
空间效率不高。
时间效率高：O(1)时间实现读写，查找。（数组实现简单的哈希表：数组的下标设为哈希表的键值（key），数组中的值设为哈希表的值（value），每个下标和对应的值构成一组“键值-值”对）

动态数组
目的：解决数组空间利用率不高的问题。
实现：先开辟较小的空间，然后网数组添加数据。当数据的数目超过数组容量时，重新开辟新的容量，大小为原来的2倍。将之前的数据复制到新数组中，再将原数组内存清空。

数组与指针

  1. 声明一个数组时，其数组名也是一个指针，指向第一个元素。
  2. sizeof(数组名)：返回数组的大小；
  3. sizeof(int* p =数组名)：返回一个指针的大小，4字节。
  4. sizeof(int 数组名)：数组名退化为同类型指针，故返回的为指针的大小，4字节。
