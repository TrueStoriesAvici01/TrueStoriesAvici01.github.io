true和false

Java不允许将一个数字作为布尔值使用，如果想要在一个布尔测试中使用一个非布尔值，则首先不需要用一个条件表达式将其转换为布尔值。

迭代

while、do-while和for用来控制循环，称为迭代语句（iteration statement）。

语句反复执行、直到起控制作用的布尔表达式（boolean expression）为“false”时终止。

do-while

do-while和while的区别：
即便布尔表达式为false，do-while中的语句至少执行一次，而while中的语句可能一次都不执行（布尔表达式为false）。

  1. 初始化（initialization）表达式、布尔表达式（Boolean-expression）或步进（step）运算都可为空。

  3. 每次循环结束，都执行一次步进。

return

两个用途

  1. 指定一个方法返回什么值。
  2. 导致当前的方法退出，并返回指定值。

当指定一个方法为非void时，则需确保有一个返回值。

break 和 continue

break：用来退出循环，不执行循环中的剩下的语句。
continue：停止执行当前的迭代，退出循环起始处，开始下一次迭代。

while(true)和for(;;)都是无穷循环。

goto

goto只是Java的一个保留字，并不使用。
Java中的使用标签和break和continue两个关键词结合实现跳转。

在Java里需要使用标签的唯一理由是：有循环嵌套存在，而且像从多层嵌套中break或continue。
