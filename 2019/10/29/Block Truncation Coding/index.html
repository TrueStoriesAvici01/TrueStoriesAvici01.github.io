Block Trunction Coding

Posted on 2019-10-29 Edited on 2019-10-28

Introduction

Block Truncation Coding (BTC)是一种灰度图像(greyscale images)的有损图像压缩。

BTC是将图像分割成块(blocks)，然后使用量化器(quantizer)降低每个块中的灰度级的数量并同时保留相同的均值(mean)和均方差(standard deviation)。

Compression produce

编码

  1. 将待处理的图像分割成多个$4 \times 4$的块
  2. 计算每个块对应的均值和均方差
  3. 对每个块进行量化操作:
    1. 根据对于块中的每个像素进行量化。$\bar{x}$为块的均值，$x(i,j)$为块中位置为(i,j)的像素。
    2. 16比特的块与对应的均值和均方差一同进行传输

解码

  1. 首先计算“a” 和 “b”:其中$\sigma$是均方差，m是一个块中的像素总数，q是量化后块中1的个数。
  2. 为了重建块，可以由已知的量化后的块对每个像素进行替换:

    将其中的所有的“1”替换成计算出的“b”的值，所有的“0”替换成“a”的值。
