Java出错汇总

Posted on 2019-10-29

Syntax error on token “;”, , expected

表现

原因：在class中直接定义类变量，再接着对其赋值。

解决：建立类变量的同时进行赋值。

The static field XXXX.x should be accessed in a static way.

表现

原因: 接口中的定义的数据域使用接口名.变量名的方式使用。

解决:

Cannot make a static reference to the non-static filed

表现

原因: main方法为static，而成员变量 s1 非static。

解决: 可将Test类中的 s1 变量改为static

The blank final field XXX may not have been initialized.

表现

原因: 接口中可以包含数据域(即，定义变量)，但是这些变量隐式的是static和final的，在实现接口的类中无法对其进行初始化，只能在接口中定义的同时进行赋值，并且在实现的类中不能对其进行修改，可视为常量。
