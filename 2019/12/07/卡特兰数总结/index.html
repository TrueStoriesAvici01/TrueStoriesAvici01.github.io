卡特兰数总结

Posted on 2019-12-07 Edited on 2019-12-11

关于卡特兰数(Catalan number)的一些小结

定义

1, 1, 2, 5, 14, 42, 132 …

递推关系

卡特兰数是满足以下规律的数列：

设$C_n$为第$n$个卡特兰数

  1. $C_n = C_0C_{n-1} + C_1C_{n-2} + … + C_{n-1}C_0$
    = $\sum_{i = 0}^{n-1}C_iC_{n-i}$
    其中$C_0 = 1$。
  2. $C_n = \binom{2n}{n} - \binom{2n}{n-1} = \binom{2n}{n}/({n+1})$
  3. $C_n = \frac{(4n-2)}{(n+1)}C_{n-1}$

特点

  • 卡特兰数的渐进增长为$C_n \sim \frac{4^n}{n^{3/2}\sqrt{\pi}}$
  • 所有的奇卡特兰数$C_n$都满足$n = 2^k - 1$ 不懂
  • 所有其他的卡特兰数都是偶数。 不懂

实际应用

从原点(0, 0)到点B (n, n)的合法路径数。

要求：

  • 只能向右或向上进行一个单位的移动。
  • 合法路径是路径未超过对角线$y = x$

图示

提示：

  • 从(0, 0)到(n, n)的所有路径总数为$\binom{2n}{n}$。
  • 不合法的路径必然会与直线$y = x + 1$有交点。将从(0, 0)到交点的路径以直线$y = x + 1$进行对称反转，可以发现：新的路径可是从(-1, 1)到(n, n)的一条路径。即所有的不合法路径与从(-1, 1)到(n, n)的路径具有一一对应的关系。
    所有不合法的路径数为从(-1, 1)到(n, n)的路径数：$\binom{2n}{n-1}$。
  • 所以合法的路径数为$\binom{2n}{n} - \binom{2n}{n-1}$。

合法01串个数：一个由n个0和n个1构成的01字符串，求解合法的01字符串的个数。

要求：

  • 合法：对于任何一个前缀而言，其0的个数不少于1的个数。

类似问题

  1. 将n个字符1, 2, … , n入栈后出栈，总共有多少种出栈的序列？
    要求：先入后出(FILO)
  2. 合法括号的种类：对于n个’(‘和n个’)’，有多少种合法的括号序列？
    要求：序列中，任何一个前缀中’(‘不少于’)’的个数

提示

将0看作向右走，1看作向上走，则此问题等价于从(0, 0)到(n, n)的路径中没有越过$y = x$对角线的路径数。

给定节点求解二叉树的个数： 由n个节点，可以形成多少个不同的二叉树？

提示

分治思想：

  1. 设n个节点的二叉树的个数为$C_n$。
  2. 先固定一个节点作为根节点，剩下n-1个节点要么在左子树，要么在右子树，则对应的种类为：$C_0C_{n-1} + C_1C_{n-2} + … + C_{n-1}C_{0}$，则$C_n = \sum_{i = 0}^{n-1}C_iC_{n-1}$。(左子树节点个数设为$C_i$)
  3. 上式是卡特兰数的一个递推关系式，所以$C_n = \binom{2n}{n} - \binom{2n}{n}$

思路2

将二叉树写成后缀表达式，则为$\lceil n/2 \rceil$个值和$n - \lceil n/2 \rceil$个运算符，其中表达式中任意一个前缀式中数值节点不少于运算符节点的个数。

凸多边形进行三角切割的方案数：求凸n+2边形用其n-1条对角线分割为互不重叠的三角形的分法总数。

提示

思路1
若将凸多边形进行分割，变成两个子凸多边形，则可以看出：
$C_n$表示为将凸n+2边形分割成三角形的总数。
则：
$C_n = C_0C_{n-1} + C_1C_{n-2} + … + C_{n-1}C_0$，为第n个卡特兰数。

思路2
待续…

圆周上有2n个点，以这些点为端点相连，形成互不相交的n条弦，其合法的连接数？

图示

提示

可以看出，将相邻两个点相连，并从圆中消去，再从圆中选择两个相邻点连接，再消去，直到所有点均消去，是合法的连接方式。
若将上面相邻两个点分别看作左右括号，则问题转换为合法括号问题。
最终的合法方案数即为卡特兰数。

利用母函数推导卡特兰数通项公式*

卡特兰数的递推式为：
$C_n = \sum_{i = 0}^{n-1}C_iC_{n-i-1} = C_0C_{n-1} + C_1C_{n-2} + … + C_{n-1}C_{0}$

其中$C_0 = C_1 = 1$。

定义母函数：
$C(x) = \sum_{i=0}^{\infty}C_ix_i = C_0 + C_1x + C_2x^2 + …$

对母函数平方可得：
$C^2(x) = (C_0 + C_1x + C_2x^2 + …)(C_0 + C_1x + C_2x^2 + …)$

$ = C_0C_0 + (C_0C_1 + C_1C_0)x + (C_0C_2 + C_1C_1 + C_2C_0)x^2 + …$

$ = \sum_{i = 0}^{\infty}(\sum_{j = 0}^{i}C_jC_{n-j})x^i$

$ = \sum_{i = 0}^{\infty}C_{i+1}x^i$

将$C^2(x)$与$C(x)$错位相减可得：
$xC^2(x) - C(x) = -C_0 = -1$

解得：
$C(x) = \frac{1\pm \sqrt{1-4x}}{2x}$

由$C(x)$定义：
$\text{lim}_{x\rightarrow 0^+}C(x) = C_0 = 1$

则：
$C(x) = \frac{1-\sqrt{1-4x}}{2x}$

插曲： 二项式定理
$(1+x)^\alpha = 1 + \sum_{n = 1}^\infty \binom{\alpha}{n}x^n$

$C(x) = \frac{1 - \sqrt{1-4x}}{2x}$

$ = \frac{1}{2x}(1-(1+\sum_{n=1}^{\infty}(-4x)^n))$

$ = 2\sum_{n = 1}^\infty \binom{1/2}{n}(-4x)^{n-1}$

$ = 2\sum_{n = 1}^\infty \frac{(1/2) \cdot (-1/2) \cdot (-3/2) \cdot … \cdot (3/2 - n)}{n!}(-4x)^{n-1}$

$ = \sum_{n=1}^\infty \frac{(1/2) \cdot (3/2) \cdot … \cdot (n - 3/2)}{n!}(4x)^{n-1}$

$ = \sum_{n = 1}^{\infty}\frac{1\cdot3\cdot5…\cdot(2n-3)}{n!}(2x)^{n-1}$

$ = \sum_{n=1}^\infty \frac{1\cdot3\cdot5…\cdot(2n-3) \cdot (n-1)!}{n!(n-1)!}(2x)^{n-1}$

$ = \sum_{n = 1}^{\infty}\frac{(2n-2)!}{n!(n-1)!}x^{n-1}$

$ = \sum_{n = 0}^{\infty}\frac{(2n)!}{n!(n+1)!}x^n$

$ = \sum_{n = 0}^{\infty}\frac{1}{n+1} \binom{2n}{n}x^n$

故：
$C_n = \frac{1}{n+1} \binom{2n}{n}$
