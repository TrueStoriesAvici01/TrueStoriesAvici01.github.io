信息论(四)-无失真信源编码

2019-12-16

无失真信源编码定理

编码器

实质

编码:对信源的原始符号按一定规则进行变换.

目的

使得变换后得到的码符号接近等概率分布,从而提高信息的传输有效性.

定义

从信源符号到码符号的一种映射.

将信源符号集中的符号$s_j,j=1,2,…,q$(或长为N的信源符号序列)变换成由基本符号$x_j,j=1,2,…r$组成的长度为$L_i$的一一对应的输出符号序列,即:
$C = \{W_1,…,W_q\}$
$s_i \leftrightarrow W_i = (x_{i_1},x_{i_2},…,x_{i_{l_i}})$
或者:
$\alpha_i = (s_{i_1}s_{i_2}…s_{i_N})\leftrightarrow W_i = (x_{i_1},x_{i_2},…,x_{i_{l_i}})$
$W_i$为码字,$l_i$为码长.

各种码的定义

二元码
码符号集为$X = \{0,1\}$,所有码字都是二元序列.

同价码
码符号集$X:\{x_1,x_2,…,x_r\}中每个码符号x所占的传输时间相同.

等长码
一组码中所有码长都相等,则为等长码.

变长码
码中码字长短不一.

非奇异码
一组码中的所有码字都不相同:
$s_i \neq s_j \leftrightarrow W_i \neq W_j$

奇异码
一组码中有相同的码字,即:
$s_i \neq s_j$,但存在$W_i = W_j$

码的N次扩展码
码C的N次扩展码是所有N个码字组成的码字序列的集合:
若码$C = \{W_1,W_2…W_q\}$
则N次扩展码为:
$B = \{B_i = \{W_{i_1},W_{i_2}…W_{i_N}\}|i_1,…,i_N = 1,…,q\}$
即: 从原码字中可重复的选取N个码字进行组合得到的新码字.

唯一可译码
若码的任意一串有限长的码符号序列,只能被唯一的译成所对应的信源符号序列,则称为唯一可译码.
唯一可译码可分为非即时码和即时码.
非即时码:接收端收到一个完整的码字后,不能立即译码,而等下一码字开始接收后才判断是否可以译码.
即时码:任意一个码字都不是其他码字的前缀部分.

基本概念

码树

若码字序列符号为m进制,可用m个符号的码树来构造码字,每个码树有一个树根A,树根有m个分支,树枝的尽头称为叶节点,中间每个节点生出的树枝的数量等于码符号的数量m,形成m进制码树.

平均码长

编码后,每个信源符号平均所需的码元个数.
信源为:$\begin{bmatrix} X\\ P(X)\end{bmatrix} = \begin{bmatrix} x_1 & x_2 & … & x_q \\ P(x_1) & P(x_2) & … & p(x_q) \end{bmatrix}$,若对应的码字长度为$k_i$,则平均码长为$\overline{k} = \sum_i P(x_i)k_i$

码率

编码后信息传输率R,指编码后每个码元载荷的信息量:
$R = \frac{H(x)}{\overline{K}}$(比特/码元)
$H(x)$: 信源熵;
$\overline{k}$: 平均码长.

编码效率

编码后每个码元的实际信息量和能承载的最大信息量的比值.

假设码元为m进制,即可取m种可能值,则每个码元所能承载的最大信息量为$\log m$比特/码元,编码效率:
$\eta = \frac{R}{R_{max}} = \frac{\frac{H(x)}{\overline{K}}}{\log m}$

克劳夫特(Kraft)不等式

唯一可译码存在的充要条件:
各码字的长度$k_i$应满足:
$\sum_{i=1}^n m^{-k_i} \leq 1$
注: m进制的码字,n个信源符号

注: 唯一可译码一定满足克劳夫特不等式,满足克拉夫特不等式的码不一定是唯一可译码.

等长码

特点

所有的码的码长都相等.

无失真信源编码条件:

  • 信源符号与码符号序列一一对应.
  • 信源的任意N次扩展也应该是唯一可译码.

注：
非等概率信源不能依据信源熵提供的长度对信源进行编码。
要依据最大信源熵提供的长度对信源进行编码。

对信源进行等长编码，需满足：
$q \leq m^{K_L}$
$K_L$：码长
$m$：码符号集中的码元数
$q$：信源符号个数。

对信源的L次扩展信源进行编码，有$q^L$个信源符号序列(长度为L)，必须满足：$q^L \leq m^{K_L}$，即：
$\frac{K_K}{L} \geq \frac{\log q}{\log m}$

渐进等分割性(AEP)和$\epsilon$典型序列

大数定理

  • 独立等分布的随机变量$X_1,X_2,…,X_n$，当n足够大时，$\frac{1}{n}\sum_iX_i$接近其数学期望值$E(X)$.
  • $S_1S_2…S_N$统计独立等同分布的随机变量，联合概率为$P(S_1S_2…S_n)$，N足够大时，$(-\log P(S_1S_2…S_N))/N$接近于信源熵H(S),$P(S_1S_2…S_n)$接近于$2^{-NH(S)}$.

渐进等分割性AEP
若随机序列$S_1S_2…S_N$中$S_i$相互统计独立并且服从同一概率分布$P(S)$,又$a_i = (s_{i_1}s_{i_2}…s_{i_N}) \in S_1…S_N$，则$-\frac{1}{N}\log p(a_i) = -\frac{1}{N} \log p(s_{i_1}s_{i_2}…s_{i_N})$以概率收敛于$H(S)$.
$\lim_{N\rightarrow \infty}P\{|\frac{I(a_i)}{N}-H(S)|< \epsilon\} = 1$

$\epsilon$典型序列
N长的序列$a_i = (s_{i_1}s_{i_2}…s_{i_N})\in S_1S_2…S_N$,对于任意小的正数$\epsilon$，满足：
$|\frac{I(a_i)}{N} - H(S)|<\epsilon$,即$|\frac{-\log P(a_i)}{N} - H(S)|< \epsilon$,则称N长的序列$a_i$为$\epsilon$典型序列，对应的集合为$G_{\epsilon N}$.
反之为非$\epsilon$典型序列,对应的集合为$\overline{G_{\epsilon N}}$.

定理：
对于任意小的正整数$\epsilon \geq 0, \delta \geq 0$,当n足够大时，则：

  1. $P(G_{\epsilon N}) > 1 - \delta,\ P(\overline{G_{\epsilon N}}) \leq \delta$.
  2. 若$\alpha_i = (a_{i1},…,a_{iN}) \in G_{\epsilon N}$，则$2^{-N(H(s) + \epsilon)} < P(a_i) < 2^{-N(H(s)-\epsilon)}$
  3. 设$||G_{\epsilon N}||$表示$G_{\epsilon N}$典型序列集中包含的$\epsilon$典型序列个数，则：
    $(1-\delta)2^{N(H(s)-\epsilon)} \leq ||G_{\epsilon N}|| \leq 2^{N(H(s) + \epsilon)}$

等长信源编码定理
一个熵为$H(S)$的离散无记忆信源，若对其N次扩展进行等长r元编码，码长为l，对于任意$\epsilon>0$,只要满足：
$\frac{l}{N} \geq \frac{H(S)+\epsilon}{\log r}$
则可以实现无失真编码，反之，若
$\frac{l}{N} \leq \frac{H(S)+\epsilon}{\log r}$
则不能实现无失真编码。

等价形式：

  • $l \log r \geq NH(S)$
    左边：长l的码符号所能载荷的最大信息量。
    右边：长为N的序列平均携带的信息量。
  • $ \frac{l \log r}{N} \geq H(S)$
    $R = \frac{l \log r}{N}$:编码信息率。

编码效率：
$\eta = \frac{H(S)}{R} = \frac{H(S)}{\frac{l \log r}{N}}$.

等长码最佳编码效率：
$\eta = \frac{H(S)}{H(S) + \epsilon} = 1,\ (\epsilon > 0)$,由$\frac{l}{N} \geq \frac{H(S) + \epsilon}{\log r}$

由$P_E=P(\overline{G_{\epsilon N}}) \leq \delta(N, \epsilon) \leq \frac{D[I(s_i)]}{N\epsilon^2}$可知，当方差$D[I(s_i)]$和$\epsilon$给定时，只要N足够大，错误率$P_E$就可以小于任意正整数$\delta$.对应的信源序列长度N满足：
$N\geq \frac{D[I(s_i)]\eta^2}{H^2(S)(1-\eta)^2\delta}$。

变长码

说明
编码后码长K是变化的，概率大的符号用短码，概率小的符号用长码。

信源为：
$\begin{bmatrix} S\\ P(S) \end{bmatrix} = \begin{bmatrix} s_1,& s_2, &…&,s_q\\ p(s_1),&p(s_2),&…&,p(s_q)\end{bmatrix}$
码字：$W_1,W_2,…,W_q$;码长为：$l_1,l_2,…l_q$。
唯一可译码：$p(W_i) = p(s_i)$
平均码长：$\overline{L} = \sum_i p(s_i)l_i$
信息传输率:平均每个码元携带的信息量：$R = \frac{H(S)}{\overline{L}}$.

紧致码(最佳码)
某一信源和码符号集若有一个唯一可译码，其平均长度$\overline{L}$小于所有其他唯一可译码的平均长度。

定理：
若一个离散无记忆信源S具有熵为H(S),并有r个码元的码符号集$ X=\{x_1,…,x_r\}$,则总可以找到一种无失真编码方法，构成唯一可译码，其平均长度满足：
$\frac{H(S)}{\log r} \leq \overline{L} < 1 + \frac{H(S)}{\log r}$
当：
等式成立的条件$\frac{r^{-l_i}}{p(s_i)} = 1$,即$l_i = -\log_r p(s_i)$时，平均码长能达到下界。（或$p(s_i) = (1/r )^{l_r}$）.

无失真变长信源编码定时(香农第一定理)
离散无记忆信源S的N次扩展信源$S^N = \{a_1,a_2,…,a_{q^N}\}$,其熵为$H(S^N)$,并有码符号$X = \{x_1,x_2,…,x_r\}$.对信源$S^N$进行编码，总可以找到一种编码方法，构成唯一可译码，使得信源S中每个信源符号所需的平均码长$\overline{L_N}$满足：
$NH(S) \leq \overline{L_N} \log r < \log r + NH(S)$
或
$\frac{H(S)}{\log r} \leq \frac{\overline{L_N}}{N} \leq \frac{1}{N} + \frac{H(S)}{\log r}$
或
$H_r(S) \leq \frac{\overline{L_N}}{N} < \frac{1}{N} + H_r(S) \ \lim_{N\rightarrow \infty}\frac{\overline{L_N}}{N} = H_r(S)$
$\overline{L_N}$:为$S^N$中每个信源符号序列的平均码长。
$H_r(S)$:r进制信源S的熵。

平稳有记忆信源
$H(S_1S_2…S_N) \leq \overline{L_N} \log r < \log r + H(S_1…S_N)$
$\frac{H(S_1S_2…S_N)}{\log r} \leq \overline{L_N} < 1+ \frac{H(S_1…S_N)}{\log r}$
$\frac{H(S_1…S_N)}{N\log r} \leq \frac{\overline{L_N}}{N} < \frac{1}{N} + \frac{H(S_1…S_N)}{N\log r}$
$\lim_{N\rightarrow \infty}\frac{\overline{L_N}}{N} < \frac{1}{\log r}\lim_{N\rightarrow \infty}\frac{H(S_1…S_N)}{N} = \frac{H_\infty(S)}{\log r}$
