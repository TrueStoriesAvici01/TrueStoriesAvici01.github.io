无失真信源编码

香农编码

离散无记忆信源：
$\begin{bmatrix}x_1 & x_2 & … & x_n \\ p(x_1) & p(x_2) & … & p(x_n) \end{bmatrix}, \sum_i p(x_i) = 1$
$l_i = \lceil \log \frac{1}{p(s_i)} \rceil,\ i = 1,2,…,q$
$\overline{L} < H_r(S)+1$,
r：码元符号数

注：
只有当码源符号概率分布按$(\frac{1}{r})^{a_i}$时，$\overline{L}$才能达到极限值$H_r(S)$.

编码步骤

  1. 按信源符号的概率从大到小的顺序排序，设$p(x_1) \geq p(x_2) \geq … \geq p(x_n)$.
  2. 设$p(x_0) = 0$,用$p_a(x_j),\ j = i + 1$表示第i个码字的累加概率$p_a(x_j) = \sum_{i = 1}^{j-1}p(x_i)$.
  3. $-\log_2 p(x_i) \leq k_i <1-\log_2 p(x_i)$.
  4. 把$p_a(x_i)$用二进制数表示，用小数点后的$k_i$位作为$x_i$的码字.

费诺编码

编码步骤

  1. 按信源符号的概率从大到小的顺序排序，设$p(x_1) \geq p(x_2) \geq … \geq p(x_n)$.
  2. 对概率按m（码符号数）进行分组，使每组概率尽可能相等.
  3. 给每个分组分配一个码元.
  4. 对每个分组重复2、3步，直到不可分为止.

哈夫曼编码

编码步骤

  1. 将信源符号按概率由大到小顺序排队.
  2. 给两个概率最小的符号各分配一个码位，将其概率相加后合并作为一个新的符号，与剩下的符号一起，再重新排序.
  3. 给缩减信源中概率最小的符号各分配一个码元.
  4. 重复步骤2、3直到概率和为1.

注

  1. 两个最小符号分配0或1是任意的，只要每次缩减信源中保持码元分配的一致性.
  2. 新符号概率与其他符号概率相同，符号次序可以任意排序，但得到的码字不同.
  3. 一般将合并的概率放前，获得较小的码方差.
  4. 进行哈夫曼编码时，为了得到码方差最小的码（各个码字长度接近相等），应使合并的信源符号位于缩减信源序列尽可能高的位置上，以减少再次合并的次数。

LZ码

设：
信源符号集$S = \{s_1,s_2,…,s_1\}$,输入信源符号序列为$u = u_1u_2…u_L \ (u_i\in S)$

编码规则

  1. 分段：将符号序列u分成小段。（分段原则：各小段尽可能短，且各不相同，分成的小段称为短语，短语总数为：$M(u)$）
  2. 依次对个短语编号：号码称为该短语的段号。
  3. 对每个短语编码：码字 = 除最后一个符号后的短语段号编码 + 最后一符号的编码。单符号的段号码码字为0.
    (如编为二进制码，则段号需要的长度为：$n = \lceil \log M(u) \rceil$,每个符号编码需要的长度为：$\lceil \log q \rceil$).
