有噪信道编码定理

背景

信道总是存在噪声，在有噪信道中进行无措传输可以达到的最大信息传输率。

信道编码对象

信源编码器输出的数字序列（信息序列）。

信源经过等长编码，码符号往往等概率分布，可认为信道编码的信源为等概率出现。

错误概率和编码规则

有噪信道的错误概率与：

  1. 信道统计特性有关，可由信道的转移矩阵描述。
  2. 错误概率与译码准则。

译码准则：
离散单符号信道的输入符号集为$A = \{a_i\},\ i = 1,2,…,r$,输出符号集$B = \{b_i\},\ j=1,2…,s$.
若对每个输出符号$b_j$都有一个确定的数$F(b_j)$使得$b_j$对应于唯一的一个输入符号$a_i$，则这样的函数为译码规则。

平均错误概率
收到$b_j$下，译码的条件正确概率为：
$P(F(b_j)|b_j) = P(a_i|b_j)$,错误概率为：$P(e|b_j) = 1- P(a_i|b_j)$

平均错误译码概率为：
$P_e = \sum_j p(b_j)P(e|b_j) = \sum_j p(b_j)P(e|b_j) = \sum_j p(b_j)(1-P(a_i|b_j))$
表示译码后平均接收一个符号所产生错误的大小。

最大后验概率准则（最小错误概率准则）

要使得$P(e|b_j)$最小，则选择$P(F(b_j)|b_j)$为最大，则当$F(b_j) = a^\star$满足：
$P(a^\star|b_j) \geq P(a_i|b_j),\ a_i \neq a^\star$。

上式化简：
$P(b_j|a^\star)P(a^\star) \geq P(b_j|a_i)P(a_i)$

在信源等概率分布时，上式为
$P(b_j|a^\star) \geq P(b_j|a_i)$.
即为最大似然译码准则：在收到一个$b_j$后，在信道矩阵的第j列，选择最大的值对应的输入符号作为译码输出。
对应的错误概率：
$P_E = \sum_{Y,X-a^\star}P(a_ib_j)$

等价形式：
$P_E = \sum_{Y,X-a^\star}P(b_j|a_i)P(a_i)$
逐列求和，出去选为译码输出元素对应的概率

平均错误概率$P_E$与译码规则有关，应该规则由信道特性决定，则平均错误概率$P_E$与信道疑义度H(X|Y)有关

费诺不等式
$H(X|Y) \leq H(P_E) + P_E\log (r-1)$

解释：
收到Y后关于X的平均不确定性分为两个部分:

  1. 是否产生$P_E$错误的不确定性$H(P_E)$.
  2. 输入符号发送而造成的最大不确定性$P_E\log(r-1)$.

汉明距离
设$\alpha_i = (a_{i_1}…a_{i_n}),\ \beta = (b_{j_1}…b_{j_n})$为两个长度为n的二元序列，则$\alpha_i$和$\beta_i$之间的汉明距离：
$D(\alpha_i,\beta_j) = \sum_k \alpha_{i_k} \oplus b_{j_k}$

在某一码C中，任意两个码字的汉明距离的最小值称为该码C的最小距离。
$d_{min} = \min \{D(C_i,C_j)\},\ C_i \neq C_j$

最小距离译码准则（最大似然译码准则）
译码规则：
选择$F(b_j) = a^\star$，使之满足：$D(\alpha^\star, \beta_j) \leq D(\alpha_i, \beta_j)\ \alpha_i \neq \alpha^\star$
就是将收到一个码字后，将其译为与之最近的输入码字。

编码选择：尽量设法使选取的M个码字中任意两个不同码字的距离尽量打。

联合$\epsilon$典型序列

联合$\epsilon$典型序列
n长的序列对(x,y)满足：

  1. X是$\epsilon$典型序列，即对于任意小的整数$\epsilon$存在n使得：
    $|1/n\log P(x) - H(X)| < \epsilon$
  2. X是$\epsilon$典型序列，即对于任意小的整数$\epsilon$存在n使得：
    $|1/n\log P(y) - H(Y)| < \epsilon$
  3. 对于任意小的正数$\epsilon$，存在n使：
    $|1/n\log P(xy) - H(XY)| < \epsilon$
    则成序列对(x,y)为联合$\epsilon$典型序列。

$G_{\epsilon n}(X)$：表示$X^n$中x的典型序列集。
$G_{\epsilon n}(Y)$：表示$Y^n$中y的典型序列集。
$G_{\epsilon n}(XY)$：表示$X^nY^n$联合空间中xy的典型序列集，即：
$G_{\epsilon n}(XY) = \{(x,y) \in X^n \times Y^n: \\ |1/n\log P(x) - H(X)| < \epsilon \\ |1/n\log P(y) - H(Y)| < \epsilon \\ |1/n\log P(xy) - H(XY)| < \epsilon \}$

联合渐进等分割性
对于任意小的正数$\epsilon \geq 0,\ \delta \geq 0$，当n足够大时，则有：

  1. $P(G_{\epsilon n}(X)) \geq 1-\delta \\ P(G_{\epsilon n}(Y)) \geq 1-\delta \\ P(G_{\epsilon n}(XY)) \geq 1-\delta $
  2. $2^{-n[H(X)+\epsilon]} < P(x) <2^{-n[H(X)-\epsilon]} \\ 2^{-n[H(Y)+\epsilon]} < P(y) <2^{-n[H(Y)-\epsilon]} \\ 2^{-n[H(XY)+\epsilon]} < P(xy) <2^{-n[H(XY)-\epsilon]}$
  3. $(1-\delta) 2^{n[H(X)-\epsilon]} \leq ||G_{\epsilon n}(X)|| \leq 2^{n[H(X)+\epsilon]} \\
    (1-\delta) 2^{n[H(Y)-\epsilon]} \leq ||G_{\epsilon n}(Y)|| \leq 2^{n[H(Y)+\epsilon]} \\
    (1-\delta) 2^{n[H(XY)-\epsilon]} \leq ||G_{\epsilon n}(XY)|| \leq 2^{n[H(XY)+\epsilon]}$

说明

  1. 在两个随机变量情况下，信源$X^n,Y^n$和联合信源$X^nY^n$都具有渐进等分割性。
  2. X，Y是无记忆信源，n长的序列中各符号是无记忆的独立同分布，因此n长的序列x,y也符合大数定律。

有噪信道编码定理

有噪信道编码定理(香农第二定理)
设离散无记忆信道$[X,P(y|x),Y]$,$P(y|x)$为信道传递概率，其信道容量为C。当消息传输率$R< C$时，只要码长n足够长，总可以在输入符号集中$X^n$找到$M(=2^{nR})$个码字组成的一组码$(2^{nR},n)$和对应的译码准则，使信道输出端的平均错误译码概率达到任意小$P_E\rightarrow 0$.

有噪信道编码逆定理
设离散无记忆信道$[X,P(y|x),Y]$,$P(y|x)$，其信道容量为C。当信息传输率$R > C$时，无论码长n多长，总找不到一种编码，使译码错误概率达到任意小。

对于限带高斯白噪声加性信道，噪声功率为$P_n$，带宽为$W$，信号平均功率受限为$P_s$，则

  1. 当$R\leq C=W\log(1+P_s/P_n)$，总可以找到一种信道编码在信道中以信息传输率（码率）R传输信息，而使平均错误概率任意小。
  2. 当$R>C$，找不到任何信道编码，在信道中以R传输信息而使平均错误概率任意小。

对于离散无记忆信道(DMC)，当$R< C$时，平均错误概率为
$P_E \leq e^{-nE_r(R)}$
$E_r(R)$为随机编码指数(可靠性函数或误差指数)
$E_r(R) = \max_{0\leq \rho \leq 1}\max_{p(X)}\{E_0[\rho, p(x)]-\rho R\}$

联合信源信道编码定理

将通信系统设计成两部分：

  1. 信源编码
  2. 信道编码

信源编码：用尽可能少的信道符号表达信源，减少冗余。
信道编码：适当增加冗余度，使能纠正和克服信道中引起的错误和干扰。

信源-信道编码定理
若$S^n = (S_1S_2..S_n)$是有限序号集的随机序列并满足渐进等分割性(APR),又信源S极限熵$H_\infty< C$则存在信源,信道编码,其$P_E\rightarrow 0$.
反之,对于任意平稳随机序列,若极限熵$H_\infty> C$,则错误概率远离0,即不可能在信道中以任意小的错误概率发送随机序列.

信源通过信道有效可靠传输的充要条件: H < C
