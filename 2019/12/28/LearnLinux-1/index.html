初始Linux

Linux是什么

计算机硬件:

  • 输入单元
  • 主机部分:(CPU, 内存, 主板)
  • 输出单元

CPU分类:

  • x86架构: 用于个人电脑,服务器等
  • ARM架构: 移动性设备.

硬件,操作系统,应用程序之间的关系:
硬件 => 内核 => 系统调用 => 应用程序
其中: 内核+系统调用 = 操作系统

GNU的GPL
GPL强调的是自由使用的权利,而非免费.

Open Source
开放源码,并且自由使用.

Linux distributions(Linux发行版)

RPM软件管理 DPKG软件管理 其他
商业 RHEL,SuSe Ubuntu
社区 CentOS,OpenSuSe Debian,B2D Gentoo

CentOS初体验

文本/图形用户界面切换

有六个终端(Terminal),
使用Ctrl + Alt + F1 ~ F6进行六个终端的切换.
F1 ~ F6对应于tty1 ~ tty6的环境操作界面.
其中 F1为图形用户界面,F2 ~ F6为文本用户界面.

提示符:
#: root用户.
$: 一般用户.

退出系统: exit.(非关机,从系统中注销)

文本命令操作

命令 含义
ls 列出文件名
ll 详细列出文件信息
ll / 列出根目录文件信息
ll / -a 显示所有(含隐藏)文件信息
clear 清屏
↑↓→← 查找历史命令,更改历史命令
history 列出所有历史命令
!数字 调用指定的历史命令
w 显示当前活跃用户
poweroff,halt,shutdown -h now, systemctl poweroff 关机
