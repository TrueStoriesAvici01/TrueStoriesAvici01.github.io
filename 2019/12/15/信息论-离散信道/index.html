离散信道及其信道容量

信道的数学模型及分类

信道: 输入集合与输出集合间的映射关系.
输入和输出是统计依赖关系

数学模型

$\{X\ P(Y|X)\ Y\}$

分类:

  1. 输入输出随机信号的特点:
  • 离散信道
  • 连续信道
  • 半离散/半连续信道
  • 波形信道
  1. 输入输出随机变量个数
  • 单符号信道
  • 离散无记忆扩展信道(多符号信道)
  1. 输入输出个数
  • 单用户信道
  • 多用户信道
  1. 信道上有无干扰
  • 有干扰信道
  • 无干扰信道
  1. 信道有无记忆特性
  • 无记忆信道
  • 有记忆信道

单符号离散信道

信道模型:

输入: $X\in \{x_1,x_2,…,x_n\}$
输出: $Y\in\{y_1,y_2,…,y_m\}$
信道模型: $p(y_j|x_i)$
其中: $\sum_jP(y_j|x_i) = 1$, $\sum_i P(x_i)P(y_j|x_i) = \sum_iP(x_iy_j) = P(y_j)$

信道统计特性:

统计特性: 信道转移概率.
信道转移概率: 条件概率$p(y_j|x_i)$
信道特性表示: 信道转移概率矩阵(信道矩阵)(第i行第j列代表输入为$x_i$,输出为$y_j$的转移概率)
反信道矩阵:条件概率$p(x_i|y_j)$

离散信道分类

  1. 无干扰信道
  2. 有干扰无记忆信道
  3. 有干扰有记忆信道

无干扰信道:
输出Y和输入X之间是一一对应的.
$y = f(x)$, $p(y|x) =\begin{cases}1 &\ y = f(x)\\ 0 &\ y \neq f(x)\end{cases}$

有干扰无记忆信道:
有干扰: 信道输出符号与输入符号之间无一一对应关系.
无记忆: 任意时刻输出符号只与对应时刻的输入符号有关.
$P(y|x) = P(y_1y_2…y_N|x_1x_2…x_N) = P(y_1|x_1)P(y_2|x_2)…P(y_N|x_N)$

有干扰无记忆信道:
处理方式:

  • 将记忆性强的N个符号看作一个矢量符号,各矢量符号之间是无记忆的.
  • 马尔科夫链.

平均互信息及平均条件互信息

平均互信息量

先验概率: 信源发出$x_i$的概率$p(x_i)$.
后验概率: 信宿收到$y_i$后推测发出$x_i$的概率:$p(x_i|y_j)$

  1. 互信息量: 后验概率与先验概率比值的对数.
    $I(x_i;y_j) = \log\frac{p(x_i|y_j)}{p(x_i)} = I(x_i) - I(x_i|y_j)$
    互信息量: 已知$y_j$后,消除$x_i$的不确定性的量.
    性质:
    对称性:$I(x_i;y_j) = I(y_j;x_i)$
    X和Y相互独立时: $I(x_i;y_j) = 0$
    互信息量可正可负可零:
    正:后验大于先验;
    负:后验小于先验;
    零:后验等于先验,两个随机事件相互独立.
  2. 条件互信息量:
    $x_i$与$y_jz_k$之间的互信息量:
    $I(x_i;y_jz_k) = \log \frac{p(x_i|y_jz_k)}{p(x_i)}$
    条件互信息: 在给定$z_k$条件下,$x_i$与$y_j$之间的互信息量:
    $I(x_i;y_j|z_k) = \log \frac{p(x_i|y_jz_k)}{p(x_i|z_k)}$
    则$x_i$和$y_jz_k$的互信息量:
    $I(x_i;y_jz_k) = I(x_i;z_k) + I(x_i;y_j|z_k)$
    说明: 联合事件$y_jz_k$发生给$x_i$提供的信息量,等于$z_k$发生后提供关于$x_i$的信息量与给定$z_k$之后再出现$y_j$所提供关于$x_i$的信息量之和.
  3. 平均互信息量
    定义
    互信息量在联合概率空间中的统计平均值（交互熵）
    $I(X;Y) = \sum_i\sum_j p(x_iy_j) = \sum_i\sum_j p(x_iy_j) \log \frac{p(x_i|y_j)}{p(x_i)}$
    物理含义
    输出端：$I(X;Y) = H(X) - H(X|Y)$
    其中$H(X)$：X的先验不确定性/无条件熵；$H(X|Y)$：信道疑义度/损失熵，收到Y后对X的后验不确定度。$I(X;Y)$：从Y获得的关于X的平均信息量。
    输入端：$I(Y;X) = H(Y) - H(Y|X)$
    其中$H(Y|X)$：噪声熵：发出X后，对Y的平均不确定度；$I(Y;X)$：发出X前后Y的先验不确定度减少量。
    系统总体：$I(X;Y) = H(X) + H(Y) - H(XY)$
    $H(XY)$：通信后，系统仍存在的不确定度。
  4. 平均条件互信息
    $I(X;Y|Z) = H(X|Z) - H(X|YZ)$
    已知Z条件下，每收到Y后获得的关于X的信息量，等于已知Z下X的平均信息量减去已知YZ同时发生，X的平均信息量。

平均互信息的特性

对称性
$I(X;Y) = I(Y;X)$

非负性
$I(X;Y) \geq 0$
平均互信息量表示从一件事件中提取另外一件事件的信息，最坏情况下为两件事件无关，从一件事件中不能得知关于另外一件事件的任何信息。

极值性
$I(X;Y) \leq H(X)\\ I(Y;X) \leq H(Y)$

凸函数性
$I(X;Y) = \sum_i\sum_j p(x_i)p(y_j|x_i) \log \frac{p(y_j|x_i)}{p(y_j)} = \sum_i\sum_j p(x_i)p(y_j|x_i) \log \frac{p(y_j|x_i)}{\sum_i p(x_i)p(y_j|x_i)}$
平均互信息量$I(X;Y)$是输入信源概率分布$p(x_i)$的上凸函数，平均互信息量$I(X;Y)$是输入转移概率分布$p(y_j|x_i)$的下凸函数
固定信道时：输入等概率分布时，接收端获得最大信息量。
固定信源时：当$p(y_j|x_i)=1/2$时，接收端获得最小的信息量。

数据处理定理
串联信道：$X\rightarrow P(Y|X)(信道1) \rightarrow Y \rightarrow P(Z|Y)(信道2) \rightarrow Z$
数据处理定理：消息经过多级处理，输入消息与输出消息之间的平均互信息量趋于变少。

信道容量及其一般计算方法

功能：以信号形式传输和存储信息。
定义：传输消息的媒介或通信。
随机性：变换是随机的。
描述：条件转移概率表示。
意义：信道中平均每个字符传达的信息量（信道的信息传输率）

定义
信道的信息传输率：$R = I(X;Y) = H(X) - H(X|Y)$(比特/符号)
是信源无条件概率$p(x_i)$和信道转移概率$p(y_j|x_i)$的二元函数：
$I(X;Y) = \sum_i \sum_j p(x_i)p(y_j|x_i) \log \frac{p(y_j|x_i)}{\sum_i p(x_i)p(y_j|x_i)}$
当信道特性$p(y_j|x_i)$固定时，存在一个$p(x_i)$，使得信道能够传送的信息率最大。

信道容量C：信道中最大的信息传输率。$C = \max_{p(x_i)}R = \max_{p(x_i)} I(X;Y)$（比特/信道符号）
单位时间的信道容量$C_t$：信道平均传输一个符号要t秒，单位时间信道容量为$C_t = 1/t \max_{p(x_i)}I(X;Y)$

离散无噪信道的信道容量

具有一一对应关系的无噪信道（无噪无损信道）

  1. 已知X或接收Y后对应的Y或X没有不确定性。
  2. $I(X;Y) = H(X) = H(Y)$
  3. 信源等概率分布时，信道达到信道容量：$C = \max_{p(x_i)}I(X;Y) = \max_{p(x_i)}H(X) = \log n$

具有扩展性的无损信道（有噪无损信道）

  1. 输入X的符号集个数小于输出Y的符号集个数。
  2. 每列只有一个非零元素，已知Y后，X不再有任何不确定度。
  3. $I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(X)$
  4. $H(X|Y) = 0,\ H(Y|X) > 0,\ H(X) < H(Y)$

具有归并性能的无噪信道（无噪有损信道）

  1. 输入X的符号集大于输出Y的符号集个数
  2. 每行只有一个非零元素，已知一个x后，对应的y完全确定。
  3. $I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(Y)$
  4. $H(Y|X) = 0,\ H(X|Y) > 0$

强对称离散信道的信道容量

概念

单符号离散信道X和Y的取值均由n个不同符号组成，每个符号的正确传递概率为：$\overline{p} = 1-p$，其他(n-1)个符号的错误传递概率为：$\frac{p}{n-1}$。

信道矩阵：
$P_{n\times n} = \begin{bmatrix} \overline{p} & \frac{p}{n-1} & … & \frac{p}{n-1}\\
\frac{p}{n-1} & \overline{p} & … & \frac{p}{n-1}\\
…\\
\frac{p}{n-1} & \frac{p}{n-1} & … & \overline{p}\end{bmatrix}$

信道矩阵的特点

$I(X;Y) = H(Y) - H(Y|X)$

$H(Y|X) = -\sum_i\sum_j p(x_i)p(y_j|x_i) \log p(y_j|x_i) = \sum_i p(x_i)H_{ni}$

$H_{ni} = -\sum_jp(y_j|x_i) \log p(y_j|x_i)$是固定$X = x_i$时对Y求和，是与X无关的常数。

$H(Y|X) = H_{ni}$

$I(X;Y) = H(Y) - H_{ni}$

$C = \max_{p(x_i)}[H(Y) - H_{ni}]$

信道容量

一种输入分布使得$H(Y)$取得最大值，由输出为n个符号，则其等概率分布时，$H(Y)$最大。

由$\begin{bmatrix} p(y_1)\\ p(y_2) \\ …\\ p(y_n)\end{bmatrix} = \begin{bmatrix} \overline{p} & \frac{p}{n-1} & … & \frac{p}{n-1}\\
\frac{p}{n-1} & \overline{p} & … & \frac{p}{n-1}\\
…\\
\frac{p}{n-1} & \frac{p}{n-1} & … & \overline{p}\end{bmatrix} \begin{bmatrix} p(x_1) \\ p(x_2) \\ … \\ p(x_n) \end{bmatrix}$

可知：当输入符号X为等概率分布时，输出Y为等概率分布，此时最大。

信道容量：
$C = \log n - H_{ni} = \log n + \overline{p}\log \overline{p} + p\log \frac{p}{n-1}$

二级制均匀信道

n=2的强对称离散信道是二级制均匀信道。
信道容量：$C = 1 + \overline{p}\log \overline{p} + p\log p = 1-H(p)$

对称离散信道的信道容量

可排列性

行（列）可排列：矩阵的每一行（列）是同一个集合的不同排列。
矩阵可排列：矩阵的行和列都是可排列的。

定义

对称离散信道：信道矩阵具有可排列性。

信道容量

$I(X;Y) = H(Y) - H(Y|X) = H(Y) + \sum_i\sum_j p(x_i)p(y_j|x_i) \log p(y_j|x_i) = H(Y) - \sum_ip(x_i) H_{mi}$

由于每一行是相同元素集的排列，则$H_{mi}$与输入X无关，则$H(Y|X) = H_{mi}$

信道容量：
$C = \max_{p(x_i)} [H(Y) - H_{mi}] = \log m - H(q_1,…,q_m)$

准对称离散信道的信道容量

定义
一个n行m列单符号离散信道矩阵，行可排列，列不可排列，矩阵m列可分成不相交的子集，子集具有可排列性。
信道容量
$C = \log r - H(q_1,q_2,…,q_m) - \sum_k N_k\log M_k$
其中：$N_k = \sum_{y\in Y_k}p(y|x_i),\ M_k = \sum_X p(y_j|x),\ y_j \in Y_k$,$N_k$是第k个子矩阵中行元素之和，$M_k$是第k个子矩阵中列元素之和。

离散信道容量的一般计算方法

离散信道容量的一般计算方法

  1. 由$\sum_j p(y_j|x_i)\beta_j = \sum_j p(y_j|x_i) \log p(y_j|x_i)$,求得$\beta_j$.
  2. 由$C = \log (\sum_i 2^{\beta_j})$,求C.
  3. 由$p(y_j) = 2^{\beta_j-C}$,求$p(y_j)$.
  4. 由$p(y_j) = \sum_i p(x_i)p(y_j|x_i)$,求$p(x_i)$.

离散无记忆扩展信道及其信道容量

定义
多符号离散信源在N个不同的时刻分别通过单符号离散信道,输出端输出随机矢量,形成一个新的信道,称为多符号离散信道.
为单符号离散信道的N次扩展信道.

N次扩展信道的输入符号数由r种扩展为$r^N$,输出符号数由s种扩展到$s^N$种.

构成N次扩展信道的传递矩阵
$P(\mathbf{Y}|\mathbf{X}) = P(Y_1Y_2…Y_N|X_1X_2…X_N)$
在输入消息为$\alpha_k$下,输出消息为$\beta_h$的传递概率:$P(\beta_h|\alpha_k) = P(b_{h_1}b_{h_2}…b_{h_N}|a_{k_1}a_{k_2}…a_{k_N})$

离散无记忆信道的N次扩展信道

若N次无记忆扩展信道的传递概率满足:
$P(\mathbf{Y}|\mathbf{X}) = P(Y_1Y_2…Y_N|X_1X_2…X_N) = \prod_k P(Y_k|X_k)$,则该单符号离散信道称为离散无记忆信道,对应的N次扩展信道称为离散无记忆信道的N次扩展信道.

离散无记忆N次扩展信道的信道容量$C^N = N \times C$

两个定理:

  1. 若信道的输入随机序列是X,通过信道传输,接收的随机序列是Y.假设信道是无记忆的,即信道转移概率满足$P(\mathbf{Y}|\mathbf{X}) = \prod_k P(Y_k|X_k)$,则存在: $I(\mathbf{X};\mathbf{Y}) \leq \sum_i I(X_i;Y_i)$
  2. 若信道的输入随机序列是X,通过信道传输,接收到的随机序列为Y,信道的传输概率为$P(Y|X)$,假设信道是有记忆的,则存在$I(\mathbf{X};\mathbf{Y}) \geq \sum_i I(X_i;Y_i)$

结论:若信源和信道都是无记忆的,则有
$I(\mathbf{X};\mathbf{Y}) = \sum_i I(X_i;Y_i)$.
对于离散无记忆信道的N次扩展信道而言,若信源是无记忆的,则有$I(\mathbf{X};\mathbf{Y}) = NI(X;Y)$.

独立并联信道及其信道容量

联合平均互信息不大于各自信道的平均互信息之和. 独立并联信道的信道容量:
$C_{1,2…N} = \max_{p(x_1x_2…x_N)}I(X_1…X_N;Y_1…Y_N) \leq \sum_i C_i$
$C_i = \max_{p(x_i)}I(X_i;Y_i)$

结论: 独立并联信道的信道容量不大于各个信道的信道容量之和,只有当输入符号相互独立时,且输入符号的概率分布达到各信道容量的最佳输入分布时,独立并联信道的信道容量才等于各信道容量之和.

串联信道的互信息和数据处理定理

若后一个信道的传递概率只与其输入有关,与前一个信道的输入无关:
$p(z|yx) = p(z|y)$,
则称这两信道的输入输出x,y,z序列构成马尔科夫链.

这两个串联信道可以等价与一个信道,输入是x,输出是z,转移概率为:
$P(z|x) = \sum_y P(y|x)P(z|xy)$

总信道的传递概率:$P(z|x) = \sum_Y P(y|x)P(z|y)$

串联信道的两个定理

  1. $I(XY;Z) \geq I(Y;Z)$,当且仅当$P(Z|XY) = P(Z|Y)$时等式成立.
  2. 若X,Y,Z组成一个马尔科夫链,则有
    $I(X;Z) \leq I(X;Y)$
    $I(X;Z) \leq I(Y;Z)$

信源与信道的匹配

定义
对于某个信源,若使得信道的信息传输率达到了信道容量,称此信源与信道达到匹配.否则认为有剩余.

信道剩余度
$ = C - I(X;Y)$

信道相对剩余度
$ = \frac{C - I(X;Y)}{C} = 1 - \frac{I(X;Y)}{C}$

无损信道的相对剩余度
$ = 1-\frac{H(X)}{\log r}$

香农无失真信源编码理论
将信源输出的信息变换成适合信道传输的新信源符号来传输.使新信源的符号接近等概率分布,新信源的熵接近最大熵,从而使信道传输的信息量达到最大,信道剩余度接近零.
