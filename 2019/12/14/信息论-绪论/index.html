信息论-绪论

2019-12-14

绪论

基本概念

信息

事物运动或存在的不确定性的描述.

信息量

与不确定性(随机性)消除的程度有关.

自信息

消息a出现的不确定性和a出现的概率成反比:
$I(a) = \log(1/p(a)) = -\log (p(a))$

互信息

接受到b后,获得关于a的信息量:
$I(a:b) = \log (1/p(a)) - \log (1/p(a|b)) = \log (p(a|b)/p(a))$

信息论的研究对象,目的和内容

通信系统模型

  1. 信源: 消息序列的来源,消息具有随机性和不确定性.
  2. 编码器: 将消息编码成信号.
  3. 信道: 消息传递的媒介.
  4. 译码器: 将信道输出的信号进行反变换.
  5. 信宿: 消息的接收方.
  6. 噪声: 信道中的干扰.

信源

分类:

  • 离散信源和连续信源
  • 无记忆信源和有记忆信源

关键:
消息中包含的信息量和消息的度量

编码器

功能:
消息变成信道传输的信号

分类:

  • 信源编码器(source encoder): 消息变成符号,提高传输有效性.
  • 信道编码器(channel encoder): 编码符号增加冗余,提高传输可靠性.
  • 调制器(modulator): 将符号变成适合信道传输的信号,提高传输效率.

信道

一种逻辑关系或通道.

分类:
无噪信道和有噪信道.

译码器

功能:
从信号中恢复消息.

包括:

  • 解调器
  • 信道译码器
  • 信源译码器

研究目的

提高信息系统的可靠性,有效性,保密性和认证性.
