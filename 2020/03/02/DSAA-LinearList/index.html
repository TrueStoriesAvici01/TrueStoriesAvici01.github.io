线性表

[toc]

线性表的定义

线性表:
由n个元素的有限序列(n>=0).

  1. 有序表可以是空集合,或者为(a1, a2,…, an).
  2. 存在唯一的第一个元素a1与存在唯一的最后一个元素an.
  3. 除了第一个元素a1外,每个元素都有唯一的前驱(predecessor).
  4. 除了最后一个元素an外,每个元素都有唯一的后继(successor).

用途:

  • 数组
  • 字符串

使用连续的存储空间(contiguous allocation)存储,编译时将内存分配给相应的变量.

线性表的抽象数据类型

操作符 返回值 含义
InitList() void 构造新的线性表
ClearList() void 将线性表置空
ListEmpty() boolean 若为空,则返回true,否则返回false
ListLength() int 返回表中数据元素的个数
GetElem(int i) Object 返回指定位置的数据元素
PriorElem(Object o) Object 若存在,返回指定元素的直接前驱
NextElem(Object o) Object 若存在,返回指定元素的直接后继
ListInsert(int i, Object o) void 将指定元素插入到指定位置
ListDelete(int i) Object o 删除指定位置的元素,并返回被删除的元素

数组

相同数据类型的变量的集合,并且在内存中占有一块连续的内存空间.
集合由n个相同类型的元素组成,且各元素之间具有线性次序.将各元素存放于起始地址为A,物理位置连续的一段存储空间,统称为数组(array).

属性:

  1. 起始地址: 数组名所在内存中的起始地址.
  2. 维数(dimension): 数组的维度.
  3. 下标的上下限: 数组中元素所在存储位置的最大值和最小值.
  4. 数组元素的个数: 下标上限和下标下限之差+1.
  5. 数组类型: 数组的类型,决定元素占用内存的大小.

特点:

  • 一般的,元素对应的下标从0开始.
  • 所有索引小于当前的元素称为其的前驱(predecessor).
  • 所有索引大于当前的元素称为其的后继(successor).
  • 当前元素的前一个元素称为其的直接前驱(intermediate predecessor).
  • 当前元素的后一个元素称为其的直接后继(intermediate successor).
  • 任一元素的所有前驱构成其前驱(prefix),所有后继构成其后缀(suffix).

注:

  • 多维数组在一维的物理内存中表示,内存地址是按照线性顺序递增的.

一维数组

声明为: A(n:m).
设起始地址为a,单个元素占用内存为d,则第i个元素对应的地址为:
a + (i-n)d. 其中 n <= i <= m.*

语法声明:
数据类型 数组名[数组长度]

  • 数据类型: 数组中存放的数据类型.可以为基本数据类型(int,float,char),或者引用数据类型
  • 数组名: 数组的名称.
  • 元素的个数: 数组可以存放的数据个数.

二维数组

声明为A(n1:n2,m1:m2).
表示(n2-n1+1)行,(m2-m1+1)列.

设元素按行排列,二维数组的起始地址为a,单个元素占用的空间为d,则元素(i,j)对应的地址为:
a + (i - n1) (m2-m1+1) d + (j - m1 + 1) * d.

声明格式:
数据类型 数组名[行数][列数]

注:

  • 行下标和列下标都是从0开始的.

多维数组

可以看作是一维数组的扩展,都是按照顺序存储的.

数组实现线性表

插入

输入: 指定的位置 i, 插入的元素 e.
输出: 无.

  1. 检查给定的位置是否合法(是否越界)
  2. 检查缓存数组是否有空闲的空间.
  3. 将指定位置后续的元素向后移动一位.
  4. 向指定的位置插入给定的元素.
  5. 更新线性表的长度.

删除

输入: 指定的位置 i.
输出: 被删除的元素 o.

  1. 检查给定的位置是否合法(是否越界).
  2. 备份被删除的元素.
  3. 将被删除的元素后续的元素向前移动一位.
  4. 将原最后一个元素置null.
  5. 更新线性表的长度.
  6. 返回被删除的元素.

合并

输入: 待合并的表 l.
输出: 无.

  1. 对给定表l中各个元素,依次插入到当前表中的末尾.

归并

输入: 待合并的线性表 l1,合并后的线性表 l.
输出: 无.

  1. 若待合并的两个表非空,则依次比较两个表中的元素.元素较小的将复制到合并表l中,并且其指针移动到下一个元素.
  2. 若当前表为空,则将表l1剩下的元素复制到合并表l中.
  3. 若表l1为空,则将表l中剩下的元素复制到合并表l中.

向量

对线性数组的抽象,不再限定同一个向量中的所有元素属于同一个类型.

ADT接口

向量ADT支持的操作接口

操作 功能
getSize() 返回向量的当前规模(元素总数)
isEmpty() 判断向量是否为空
getAtRank(r) 获得索引为r的元素
replaceAtRank(r,e) 用e将索引为r的元素替换
insertAtRank(r,e) 将e插入索引r处,并返回该元素
removeAtRank(r) 删除索引为r的元素,后继元素向前移动,返回被删除的元素

矩阵

矩阵可以用二维数组表示.

稀疏矩阵

一个矩阵中绝大多数元素为0.

存储:

  • 一般使用二维数组用来存储非零项,节省空间.
  • 每一行代表一个非零项,分别有对应的行,列以及存储的数据.
