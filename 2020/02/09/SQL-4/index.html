注:

  • 按照过滤条件对结果进行过滤.
  • WHERE子句的位置在ORDER BY之前,WHERE之后.

WHERE 子句操作符

WHERE子句操作符

操作符 说明 操作符 说明
= 等于 > 大于
< > 不等于 >= 大于等于
!= 不等于 !> 不大于
< 小于 BETWEEN 在指定的两个值之间
<= 小于等于 IS NULL 为NULL值
!< 不小于
