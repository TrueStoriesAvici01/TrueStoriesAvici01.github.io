数据库基础

数据库

数据库(database):
保存有组织的数据的容器.

注意区别数据库和数据库软件,数据库软件称为数据库管理系统(DBMS).

表

表(table):
某种特定类型数据的结构化清单.

存储在表中的数据是同一种类型的数据或清单.

表名:
每个表使用一个名字标识自己.

  • 使用数据库名和表名等组合实现表名唯一性.
  • 同一个数据库不能有两个相同表名的表.
  • 不同的数据库可以使用相同的表名.
  • 表的特性定义了数据的存储,分解,各部分命名.
  • 模式(schema)是关于数据库和表的布局及表的布局及特性的信息.

列和数据类型

列(column):
表中的一个字段,所有表都是由多列组成.
列存储表中某部分信息.

数据类型(datatype):
定义列可以存储的数据种类.(数值,字符…)

  • 每个表列都有相应的数据类型,其限制该列中存储的数据.
  • 数据类型及其名称时SQL不兼容的一个主要原因.

行

行(row):
表中的一个记录.

数据库记录(record)为列的别称.

主键

主键(primary key):
一列(或一组列),其值能够唯一标识表中每一行.

要求:

  • 任意两行不具有相同的主键值.
  • 每一行具有一个主键值(不允许为null).
  • 主键值不允许修改或更新.
  • 主键值不能重用.(行删除后,其主键不能赋给新行)

可以使用多个列作为主键,则单列可以不唯一,但所有列组合唯一.

什么是SQL

SQL:
SQL: Structured Query Language(结构化查询语言)的缩写,是用来与数据库沟通的语言.
