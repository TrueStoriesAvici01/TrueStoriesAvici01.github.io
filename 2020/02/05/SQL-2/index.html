检索数据

[toc]

SELECT语句

关键字(keyword):

  • 作为SQL组成部分的保留字.
  • 不能作为表或列的名字.
  • 每个SQL语句由一个或多个关键字构成.

检索单个列

注:

  • 返回结果为未排序数据.
  • 使用分号(;)分隔不同SQL语句.
  • SQL语句不区分大小写(关键字一般大写,易于阅读)
  • 所有空格被忽略,可以写成多行.

检索多个列

注:

  • 不同列之间加上逗号(,)分隔,最后一个列名不加逗号.
  • SQL一般返回原始的,无格式的数据.

检索所有列

注:

  • 列名位置使用星号(*)实现检索所有列.
  • 可能降低检索性能.
  • 可以检索名字未知的列.

检索不同的值

注:

  • DISTINCT表示只返回不同的值.
  • DISTINCT只能位于所有列名的最前面,其前不能有列名.
  • 其后使用多列时,只有当指定列的值都相同的行才算相同,只返回一次.只要有一列不同,则认为两行不同,将都返回.

限制结果

注:

  • 结果中前n项返回.

注:

  • LIMIT:其后为输出的行个数.
  • OFFSET:其后为起始行.
  • 行的序号是从0开始的,即第0行,第1行…
  • MySQL和MariaDB:LIMIT n,m表示LIMIT n OFFSET m.
