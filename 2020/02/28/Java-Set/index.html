Java Set接口小结

简介

  • Set中没有重复的元素,并且最多一个null元素.
  • 当set的元素为可修改对象时,需要特别注意.
  • 注意对于null或不同类型的对象的处理,可能需要抛出异常.

方法总结

返回值 方法 作用
int size() 返回set中的元素个数.若超过最大值,则返回最大值
boolean isEmpty() 若set中没有元素,则返回true
boolean contains(Object o) 若set中有指定的对象,则返回true
Iterator< E > iterator() 返回set元素的一个迭代器
Object[] toArray() 返回包含set所有元素的数组
< T > T[] toArray(T[] a 返回包含set中所有元素的数组
boolean add(E e) 将指定的元素加入set中,并返回true.若已存在,则返回false
boolean remove(Object o) 若set中包含指定元素,将其删除并返回true.若不存在,则返回false
boolean containsAll(Collection<?> c) 若set中包含指定集中的所有元素,则返回true
boolean addAll(Collection<? extends E> c 将指定集中的所有元素加入set中,并返回true
boolean retainAll(Collection<?> c 移出set中所有非指定集中的元素,并返回true
boolean removeAll(Collection<?> c 移出set中指定集中所有的元素,并返回true
void clear() 移出set中所有元素
boolean equals(Object o) 若set与指定集中的所有的元素相同,则返回true
int hashCode() 返回set的哈希码值.set的哈希码值为其中每个元素的哈希码值之和
default Spliterator< E > spliterator()` 对set的元素生成一个可分隔迭代器
