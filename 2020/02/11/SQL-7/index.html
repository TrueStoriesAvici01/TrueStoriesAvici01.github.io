创建计算字段

计算字段

字段(field):即数据库的列.

计算字段:通过对数据库表中的列进行计算获得的新字段.

目的:将数据库表进行处理获得需要的格式.

拼接字段

拼接(concatenate):将值连接到一起构成单个值.

  • +: Access 和 SQL Server使用.
  • ||: DB2, Oracle, PostgreSQL, SQLite使用.
  • Concat(,,): MySQL和MariaDB使用.

注:

  • 一些数据库将数据填充成固定列宽,可使用RTRIM()去掉右边空格,LTRIM()去掉左边空格,TRIM()去掉两边空格.
  • 计算字段作为值无法重复使用,可以通过别名给计算字段命名,从而重复使用.
  • AS将其前面的列重命名为后面的列名,优先级低于拼接+.
  • AS也可对将原列名重新命名.
  • 别名也称导出列(derived column),表示为同一意思.
