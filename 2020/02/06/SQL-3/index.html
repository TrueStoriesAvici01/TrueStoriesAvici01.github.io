排序数据

子句(clause):
SQL语句由子句构成,有些是必需的,有些是可选的.子句由一个关键字加上所提供的数据组成.

注:

  • 按照特定列名的数据对结果排序.
  • ORDER BY子句为最后一条子句,否则出错.
  • ORDER BY其后可用非检索的列对数据进行排序.

按多个列排序

注:

  • 返回结果按照多列进行排序.
  • 按照顺序依次按照不同列名对结果进行排序:先按照列名1,再按照列名2…
  • 只有在前一列具有相同的值,才按照后一列对结果继续排序.

按列位置排序

注:

  • 序号:为选择列的相对位置,按照该列顺序对结果排序.
  • 若选择的列的相对位置更改,则序号也需要更改.
  • 无法用于没有被选中的列.

指定排序方向

注:

  • 在需要降序的列名后加上DESC,其表示DESCENDING(可用).
  • 若对多列降序排列,则每个列名后都使用DESC.
  • 升序为ASC(ASCENDING),默认为升序,无需指定.

大小写排序: 在字典(dictonary)排序顺序中,大小写默认相同.
