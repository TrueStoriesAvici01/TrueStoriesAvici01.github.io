树

基本概念

定义

树至少包含一个结点,并满足下列条件:

  1. 树中有个指定的结点,称为根(树根)
  2. 剩下的结点划分成n个互不相交的集合,每个集合内是一颗树(根的子树)

术语

  1. 树中的每个结点即包含信息域(存储信息)又包含指针域(与其他结点链接).
  2. 度: 每个结点的子树的个数.
  3. 叶子结点(叶子/终端结点): 度为0的结点(没有子树的结点)
  4. 非终端结点: 度非0的结点
  5. 孩子/父亲: 某个结点的子树的根结点为该结点的孩子,该结点是孩子结点的父亲.
  6. 兄弟: 同一个父亲的各个结点.
  7. 树度: 树中所有结点的度的最大值.
  8. 祖先: 从根结点到该结点的父节点所经过的所有结点.
  9. 树根位于第1层.
  10. 树的高度(深度): 树中最大层结点的层数.

表示方法

  • 链式表示
  • 左孩子右兄弟表示

链式表示

每个结点中设置

  • 一个数据域
  • k个链域

为k-度树的结点.
浪费大量存储空间

左孩子右兄弟表示

每个结点包含:

  • 一个数据域
  • 两个链域(左:孩子域;右:兄弟域)

二叉树表示

每个结点:

  • 一个数据域
  • 两个链域(左孩子,右孩子)

二叉树

定义

二叉树或是结点个数为0的空树,或是结点个数有限的结点集合,结点中有一个根结点,结点有两棵不相交的二叉子树,分别为左子树和右子树.

注:

  • 结点的度不超过2.
  • 左右子树有序,顺序为左前右后.
  • 结点个数可以为0.

性质

  • 二叉树第 i 层的最大结点个数是 $2^{i-1},\ i\geq 1$.
  • 深度为 k 的二叉树的最大结点个数是 $2^k-1, k \geq 1$.
  • 对任何非空二叉树T,令$n_0$是叶结点个数,$n_2$是度为2的结点个数,有$n_0 = n_2 + 1$.
  • 深度$k\geq 0$的满二叉树是结点个数为$2^k-1$的二叉树.

表示

顺序表示

使用一维数组存储,序号为 i 的结点存入数组的第 i 个位置.
对给定结点序号为 $i,\ 1 \leq i \leq n$,有:

  • 当 $1 < i \leq n$, i 号结点的父结点在$P(i) = \lfloor i/2 \rfloor$;当 i = 1时,该结点为根,无父结点.
  • 当 $2i \leq n$,i号结点的左孩子结点位置是 $l(i) = 2i$;当 $2i > n$时, i号结点无左孩子.
  • 当 $2i+1 \leq n$, i号结点的右孩子结点位置是 $r(i) = 2i+1$; 当$2i+1 > n$时, i号结点无又孩子.

缺陷:
对于完全二叉树适用,但是一般的二叉树会浪费存储空间.

链式表示

每个结点由下组成:

  • 一个数据域
  • 两个链域(左孩子和右孩子)

遍历

分类
V为当前结点, L为V的左子结点, R为V的右子结点

  1. 先序遍历: V => L => R
  2. 中序遍历: L => V => R
  3. 后序遍历: L => R => V
  4. 层序遍历

线索二叉树

定义

在二叉树中空链域中存放指针,指向树中的其他结点,这些指针为线索.

线索建立规则:

  • 若结点的左孩子为空,则左孩子指向指向中序遍历次序中该结点的前驱结点.
  • 若结点的右孩子为空,则右孩子指向中序遍历次序中该结点的后驱结点.

为了区分线索和链域,使用两个标志域:

  • leftThread: 若结点的左孩子为线索,则为true,否则为false.
  • rightThread: 若结点的右孩子为线索,则为true,否则为false.

大根堆

定义
每个结点都不小于孩子结点(若存在).

注:

  • 大根堆是完全二叉树.
  • 适合使用数组或链表存储.

操作

插入:

  • 将新结点插入到树的最后位置.
  • 从插入结点开始,检查其与其父的大小关系,若不满足大根树性质,则对掉位置.
  • 向上检查,父子之间大小关系,直到根结点为止.

删除:

  • 将树中最后一个结点插入到树根,删除最后一个结点.
  • 从树根开始,检查其与孩子结点的大小关系,若不满足大根树性质,则将当前结点与其孩子中最大的结点对掉.
  • 从对掉的孩子结点向下开始检查,直到叶子结点.

二叉查找树

定义

是一棵二叉树,可以是空树,或则满足:

  • 树中每个结点有一个唯一的key.
  • 如果有左子树,则左子树的所有key都小于根的key.
  • 如果有右子树,则右子树的所有key都小于根的key.
  • 左右子树都是二叉查找树.

操作

  • 查找
  • 插入
  • 删除
  • 合并与分裂

查找

  • 若为空树,则查找失败.
  • 若非空树,则检查根结点. 若等于根结点,则查找成功. 若小于根结点,则检查其左子结点; 若大于根结点,则检查右子结点.
  • 若子结点的左右孩子结点为空,则查找失败,否则继续向下查找.

插入

  • 若为空树,则直接插入为根结点.
  • 若非空树,则查找需要插入的位置.
  • 从根结点开始进行比较: 若大于根结点,则检查其右子结点;若小于根结点,则检查其左子结点;若相等,则不进行插入.
  • 循环检查,直到满足: 小于(大于)某个结点,并且该结点的左子(右子)结点为null,则在左子(右子)结点插入该结点.

删除

  • 若删除的结点为空,则直接返回.
  • 若删除的结点非空,则首先查找其父结点的位置.
  • 根据待删除结点的子结点分类处理:
  • 若待删除结点无子结点,则将其父结点置空.
  • 若待删除结点有右子结点,则用右子结点替换待删除结点,并将待删除结点的左子树置于其右子树的最左端结点的左子孩子处.
  • 若待删除结点只有左子结点,直接用左子结点替换待删除结点.

合并

  • 两棵树中一棵树中的值都小于指定值,另外一颗树中的值都大于指定值.
  • 取较大值树中的最小值为根结点,并将其从树中删除.
  • 设根的左孩子为较小值的树,右孩子为较大值的树.

问题:

要将k路归并成一组有序序列(一路).

优胜树

定义
是完全二叉树,每个结点的取值是两个孩子的最小值.
每个叶子结点指向一路序列.

排序

  • 相邻的叶子结点两两比较,取其小者的值作为两个结点的父结点.
  • 对同一层中的结点依次两两比较,取小者作为两者的父结点.
  • 合并结点,一直到根结点结束,则根结点为所有路序列的的最小者.
  • 删除叶子结点中的最小者,将其指向同一路序列的下一结点.
  • 从新结点开始,向上比较父子结点,重新确定优胜树.
  • 直到所有路都空,则排序完成.

森林

定义
是n >= 0棵不相交树的集合.

森林转化为二叉树

定义
森林是由树T1,…,Tn组成,对应的二叉树为B(T1,…,Tn),满足:

  • 若 n = 0,则B是空树.
  • 若n不等于0,则B的根是T1的根;根的左子树是B(T11,T12,…,T1m),其中T11,T12,…,T1m是T1的子树;根的右子树是B(T2,…,Tn).

注:
即将一棵树的树根的子树按照递归的方式放置在其左子结点处.第一棵子树的根作为左子结点,第二棵子树作为第一棵子树的右子结点,第三棵子树作为第二棵子树的右子结点…
对于子树,按照同样的方式: 在第一棵子树的子树中,第一棵子树作为左子结点,第二棵子树作为第一棵子树的右子结点….

遍历森林

先序遍历
先序遍历森林与先序遍历转换的二叉树相同

  1. 若空则返回.
  2. 访问第一棵子树的树根.
  3. 森林先序遍历第一棵子树的所有子树.
  4. 森林先序遍历除第一棵子树外的子树.

中序遍历
中序遍历森林与中序遍历转换的二叉树相同

  1. 若空则返回.
  2. 森林先序遍历第一棵子树的所有子树.
  3. 访问第一棵子树的树根.
  4. 森林中序遍历除第一棵子树的所有子树.

后序遍历
后序遍历森林与后序遍历转换的二叉树不同

  1. 若空则返回.
  2. 森林后续遍历第一棵子树的所有子树.
  3. 森林后续遍历除第一棵子树外的所有子树.
  4. 访问第一棵子树的树根.

层序遍历
层序遍历森林与层序遍历转换的二叉树不同

从每棵树的树根开始,按层序从左到右一一访问每个结点.

不相交集合的表示

使用树表示不相交的集合.
链方向: 从孩子结点指向父结点.

模型:
同一个集合中的元素指向同一个元素,即为该集合对应的树的树根.
合并: 让一个树的树根的父链域指向另一个树的树根.
每个集合中的树的树根指向该集合的名称.

实现:
使用数组进行存储,其中第i个元素代表元素i.
每个数组元素中存储其父结点的索引.
根结点对应的数组元素存储的是-1,用来标记其为根结点.

操作:

  • 不相交集合的合并: 将两个不相交的集合合并成一个集合,返回一个合并后大的集合.
  • 查找: 在集合中查找给定的元素,返回其所属的集合.

合并

方法一:
将第一棵树的树根的父结点改为第二棵树的树根.

方法二:
方法一会导致蜕化树,可通过加权解决
如果树i的结点个数小于树j的结点个数,则j称为i的父亲;否则i称为j的父亲.(将结点的个数的负值存储在根结点父亲链域)

查找

方法一:
从要查找的元素开始,检查其父结点位置,再到对应的位置检查其父结点,直到遇到某个结点的父结点为-1.此时的结点的位置即为要查找所属的集合.(用树根的位置标记不同的集合)

矩阵乘法

问题
n个矩阵相乘,对应的次序有多少种?

求解:
B{n}代表n个矩阵相乘的次序数.
则有:
B{n} = B{1}B{n-1} + B{2}B{n-2} +…+ B{n-1}B{1}

求解可得:
B{n} = 1/(n+1)C(2n,n)
