纠错码父类及基本概念

线性分组码

基本概念

线性分组码的编码

  • 按一定长度将信息序列分成若干码组,每组由k位组成
  • 按线性规则将各码组变换成n重码字

码字数
信息码组长k位,有$2^k$个不同的码组,即$2^k$个码字.

(n,k)线性码
信息位长为k,码长为n的线性码.

编码效率(编码速率/码率/传信率)
$R = k/n$,信道的利用效率.

一致监督方程
在k个信息码元之后附加r个监督码元,使每个监督元是其中某些信息元的模2和.

如:
对于(n,k)线性分组码,前k个是信息元,后(r = n-k)个是监督元.

一致监督方程/一致校验方程:确定信息元得到监督元规则的一组方程.

一致监督矩阵
将监督方程写成矩阵形式,有
$H\cdot C^T = 0^T$
其中:H为$r\times n$的一致监督矩阵, C为码字行向量.
标准形式:
将监督矩阵的后r列化成单位子矩阵.

特性:

  • 每一行为一个监督方程
  • r个监督方程是线性独立的,秩为r

线性分组码的生成矩阵
特性:

  • 线性码任意两个码字之和仍是一个码字.
  • 线性码由其生成矩阵或校验矩阵确定
  • 含有零码字(码字群的单位元)
  • 所有码字可由其中一组k个独立码字线性组合而成
  • 码的最小距离等于非零码的最小重量

生成矩阵:
定义:
由矩阵G生成的(n,k)线性码,矩阵G为(n,k)线性码的生成矩阵.

$G_{k\times n} = \begin{bmatrix} g_1 \\ … \\ g_k \end{bmatrix} = \begin{bmatrix} g_{11} \ …\ g_{1n}\\ … \\ g_{k1}\ … \ g_{kn} \end{bmatrix}$

特征:

  • 每一行为一个码字
  • 每个信息组m可由G求得对应的码字

生成矩阵与一致监督矩阵的关系:
线性系统码的监督矩阵H和生成矩阵G之间可以直接呼唤
$G\cdot H^T = [I_k\ Q_{k\times r}][P_{k\times k}\ I_r] = [I_k\ Q_{k\times r}]\begin{bmatrix} P_{r\times k}^T \\ I_r\end{bmatrix} = P_{r\times k}^T + Q_{K\times r} = 0$

$Q_{k\times r} = P_{r\times k}^T$

  • $G = [I_k\ Q_{k\times r}]$
  • $H = [Q_{k\times r}^T\ I]$

对偶码:
将G作为监督矩阵,H作为生成矩阵,则新的码为原来码的对偶码,是一个(n,n-k)的线性码

线性分组码的最小距离,检错和纠错能力

汉明距离:
两个码字U,V之间对应码元位上符号取值不同的个数.
$d(U,V) = \sum_i (u_i \oplus v_i)$
特性:

  • 非负性: $d(U,V) \geq 0$
  • 对称性: $d(U,V) = d(V,U)$
  • 三角不等式: $d(U,V) + d(V,W) \geq d(U,W)$

最小距离:
线性码码字集合中,任意两个码字之间的距离最小值.
$d_{min} = \min_{i\neq j}\{C^i,C^j\}$

汉明球:
以码字C为中心,半径为t的汉明球是与C的汉明距离小于等于t的所有向量.

汉明重量W:
码字中非0码元符号的个数.(二元线性码中,码字中含”1”的个数)

最小重量$W_{min}$:
非0码字重量最小值.

线性分组码的最小距离等于其最小重量

检错能力: 一个线性码能检出长度$\leq l$个码元的任何错误图样.

纠错能力: 一个线性码能纠正长度$\leq t$个码元的任意错误图样.

最小距离和纠错能力:
(n,k)线性码能够纠t个错误的充要条件是码的最小距离为:
$d_{min} \geq 2t+1$

最小距离与检错能力:
(n,k)线性码能够发现l个错误的充要条件是码的最小距离为:
$d_{min}\geq l+1$

最小距离与检,纠错能力:
(n,k)线性码能纠t个错误,并发现l个错误的充要条件是码的最小距离为:
$d_{min} \geq t+l+1$

线性码的最小距离与监督矩阵的关系:

  • 设H为(n,k)线性码的一致监督矩阵,若H中任意S列线性无关,而H中存在(S+1)列线性相关,则码的最小距离为(S+1).
  • 若码的最小距离为(S+1),则该码的监督矩阵的任意S列线性无关,则必存在有相关的(S+1)列.
  • 在二元线性码的监督矩阵H中,如果任一列都不是全0,且任两列都不相等,则该码能纠正一个错误.

译码
使用监督矩阵译码:
接收到一个码字R,检验$H\cdot R^T = 0^T$是否成立:

  • 成立,则R为一个码字
  • 不成立,码字出错

伴随式
$S = R\cdot H^T$

发送码字:C
错误图样:E ($e_i$为1则第i位出错)
接收码字:R = C + E
伴随字$S^T = H\cdot E^T$

伴随式与错误:

  • 若S = 0, 则无错
  • 若S $\neq$ 0,则有错

二元码中,伴随式是H中与错误码元对应列之和

最佳译码准则(最大似然译码)
对于FEC方式,采用纠错码后的码字差错概率为:
$p_{we} = \sum p(C)p(C’\neq C|R)$
P(C): 发送码字C的先验概率
P(C|R): 后验概率

对于BSC信道,最小化P(C’$\neq$C|R)等价与最大化P(R|C),等价于最小化d(R,C).
使差错概率最小的译码是使接收向量R与输出码字C’距离最小的译码
$C’ = \min_{C_i} d(R,C_i) = d(R,C’)$

查表译码法
对$2^k$个码字的(n,k)线性码,找到与R有最小距离的码字,可使用标准阵列.

标准阵列构造方法:

  • 先将$2^k$个码字排成一行,作为标准阵列的第一行,全0码字在最左边.
  • 剩下的$2^n - 2^k$个码字中选择重量最轻的码字$E_2$放在$C_1$下面,将$E_2$分别与第一行码字相加,作为对应位置的值.
  • 在剩下的码字中选择重量最轻的码字$E_3$,重复进行,知道所有码字用完.

性质:

  • 在标准整列的同一行中没有相同的矢量,而$2^n$个码字在阵列中有且只出现一次.
  • (线性码纠错极限定理):二元(n,k)线性码能纠$2^{n-k}$个错误图样.
  • 一个可纠t个错误的线性码必须满足:$2^{n-k}\geq \sum_i \binom ni$
  • 标准阵列译码 = 最小距离译码 = 最佳译码
  • 在标准阵列中,一个陪集的所有$2^k$个码字有相同的伴随式,不同的陪集伴随式互不相同.

(n,k)线性码的译码步骤:

  • 计算接收矢量R的伴随式$S^T = HR^T$
  • 根据伴随式和错误图样的一一对应关系,利用伴随式译码表,由伴随式得到错误图样E
  • 将接收的码字减去错误图样,得到发送码字的估值:C’ = R - E.

完备码:$2^{n-k} = \sum_i \binom ni$
