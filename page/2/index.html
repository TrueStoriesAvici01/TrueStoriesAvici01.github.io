Git理论基础

Posted on 2019-10-29 Edited on 2019-10-28

Git记录的是什么

  1. Git是将每个版本独立保存
  2. 工作区域、暂存区域和Git仓库

    工作区域 (Working Directory)：日常编程时的工作目录
    暂存区域 (Stage (Index)): 保存改动的目录
    Git仓库 (Repository(HEAD)): 最终存放所有数据的位置

Git 的工作流程

  1. 在工作目录中添加、修改文件
  2. 将需要进行版本管理的文件存放到暂存区域
  3. 将暂存区域的文件提交到Git仓库

Git 管理的文件有三个状态

  • 已修改 (modified)
  • 已暂存 (staged)
  • 已提交 (committed)

使用Matlab在训练分类模型

Posted on 2019-10-20 Edited on 2019-10-28

简介

Matlab中的Classification Learner App中包含了多种分类模型：decision tree, discriminant analysis, support vector machines, logistic regression, nearest neighbors, ensemble classification.

Automated Classifier Training

  1. 在Apps标签下，选择Math, Statistics and Optimization，点击Classification Learner。
  2. 点击New Session，从本地文件或workspace中导入数据。指定response变量和作为predictors变量。
  3. 在Classification Learner标签页中，选择Classifier，点击All Quick-To-Train。然后会自动训练所有模型。
  4. 点击Train进行训练。

    最终结果Accuracy将在训练完成后展示在History标签页中。

  5. 点击History标签页中的一个模型，探究所得结果。
  6. (optional)若要尝试所有分类器，可选All，然后点击Train进行训练。

Java编程思想(三)：操作符

Posted on 2019-10-10 Edited on 2019-10-28

Java操作符

操作符接受一个或多个参数，并生成一个新值。
有些操作符可能改变操作数自身的值。
“=”，“==”，“!=”能够操作所有对象，其他的操作符只能操作“基本类型”。
String类支持“+”和“+=”。

优先级

先乘除后加减。
使用括号明确计算顺序。
“+”可能涉及“字符串转换”：当一个String后面有“+”，其后又有非String类型元素，会试图将非String类型的元素转换为String。
